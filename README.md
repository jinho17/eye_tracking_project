# eye_tracking_project
프로젝트 진행기간
-----------------
  2020.04 ~ 2020.06  


프로젝트 주제
-----------------
  eye tracking 시선 추적 후, 사용자의 시선정보를 데이터 베이스로 추출한다.  
  시선정보의 분석값은 엑셀파일, heatmap으로 변환하여 사용자가 쉽게 이해하고 사용할 수 있도록 한다.
